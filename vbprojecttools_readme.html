vBulletin Project Tools Introduction

Thank you for using vBulletin Project Tools Open Source! We hope you enjoy our product!

Note: you must be running vBulletin 3.7.0 or newer to install vBulletin Project Tools!

Installation & Upgrading

vBulletin Project Tools is very easy to install or upgrade. All you need to do is upload the included files and import the XML via the Administrators' Control Panel.

  1. Upload the contents of the upload/ directory in this zip file over top of your existing vBulletin directory. This is the directory that contains forumdisplay.php and many other files in it.
  2. Login to the Administrators' Control Panel for your forum.
  3. Go to Plugins & Products > Manage Products > Add/Import Product
  4. You will be using the Import Product table. In "Import the XML file from your server", enter:
    ./includes/xml/product-vbprojecttools.xml
    Now, change "Allow Overwrite" to yes and click "Import".

You should now see a message saying that your install or upgrade was successful. That's it! vBulletin Project Tools is now installed!

If you have more installation questions, please see this page of the manual.
